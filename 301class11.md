# Mongo and Mongoose

SQL vs. NoSQL 

**SQL** | **NoSQL** | 
--- | --- | 
Relational Databases | Non-relational/distributed databases | 
Predefined Schema | Dynamic Schema | 
Structured query language | Unstructured query language | 
Vertically scalable | Horizontally scalable |
